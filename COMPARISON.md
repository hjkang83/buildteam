# ChatGPT vs 우리 시스템 비교 시연 자료

> PREMORTEM 시나리오 6 대응: "ChatGPT랑 뭐가 달라요?"
> 같은 질문을 ChatGPT에 물어본 응답 vs 우리 시스템의 3인 토론을 나란히 비교

\---

## 질문: "강남 오피스텔 수익률 3%면 낮은 거 아냐?"

### ChatGPT 응답 (단일 AI)

```
네, 강남 오피스텔 수익률 3%는 상대적으로 낮은 편입니다. 일반적으로 오피스텔의 
평균 수익률은 4~5% 정도로 알려져 있으며, 3%는 이보다 낮습니다. 다만, 강남은 
입지 프리미엄이 있어 시세차익을 기대할 수 있고, 공실 리스크가 낮은 편이라는 
장점이 있습니다. 투자 목적(월세 수익 vs 시세차익)에 따라 판단이 달라질 수 
있습니다.
```

**문제점:**
- ❌ "4~5% 정도로 알려져 있으며" → 출처 없음. 어디서 나온 수치인지 알 수 없음
- ❌ "시세차익을 기대할 수 있고" → 근거 없는 추측
- ❌ 대출 레버리지 효과, 취득세 영향, 실질 수익률 계산 없음
- ❌ 리스크 분석 없음 (공실률 상승, 금리 변동, 신규 공급)
- ❌ 내 상황(예산, 투자 유형, 리스크 허용도)에 맞는 자문 없음
- ❌ 대화 종료 후 정리·요약 없음

\---

### 우리 시스템 응답 (3인 C-suite 토론)

#### 💼 CFO (재무총괄)
```
대표님, 숫자만 놓고 봅시다. 강남 오피스텔 표면 수익률 3%는 관리비·공실·세금 
빼면 실질 수익률 1.8~2.2% 구간입니다 [출처: 한국부동산원 오피스텔 수익률 통계, 
2025]. 대출 60% 끼면 레버리지 수익률 4.2% 찍히지만, 취득세 4.6% 떼면 첫해엔 
마이너스예요 [출처: 지방세법 시행령]. 같은 4.5억을 예금에 넣으면 연 3.5% 
확정인데, 그 비교가 먼저입니다.
```

#### 🛡 CSO (전략총괄)
```
잠깐요, 수익률 3%보다 더 중요한 게 있습니다. 고금리 국면에서 오피스텔 진입은 
타이밍이 안 맞습니다 — 기준금리 3.5%에 대출 이자가 수익률을 잡아먹는 구간이에요 
[출처: 한국은행 기준금리, 2025.04]. 거기에 강남 오피스텔 신규 공급이 
2025~2026년 3,200실 예정이라 공실률 상승 압력도 있습니다 [출처: 부동산114]. 
하반기 금리 인하 시그널 나올 때까지 6개월 대기가 전략적으로 맞아요.
```

#### 🧭 투자컨설턴트 (투자자문)
```
대표님 상황에서 보면, 월세 수익이 목적이라면 표면 3%짜리 강남 오피스텔은 
적합하지 않습니다. 같은 4.5억 예산이면 성동·마곡 권역에서 표면 4~4.5%대 매물을 
2건 분산 매수하는 게 현금흐름과 리스크 분산 양쪽에서 유리합니다. 먼저 월세형인지 
시세차익형인지 투자 유형 기준을 정하시는 게 순서입니다.
```

#### 📋 비서실장 (자동 회의록)
```
[회의 요약]
- CFO 의견: 표면 3%에서 실질 1.8~2.2%, 예금 대비 불리
- CSO 의견: 고금리+신규공급으로 타이밍 부적합, 6개월 대기 권장
- 투자컨설턴트 의견: 강남 단일보다 중소형 분산 매수 적합
- 결정사항: 강남 오피스텔 즉시 투자 보류
- Next Action: 투자 유형 기준 정리, 성수·마곡 급매물 리스트업
```

## 차별점 비교표

| 항목 | ChatGPT | 우리 시스템 |
|-|-|-|
| **데이터 출처** | 없음 ("~로 알려져 있으며") | 국토교통부 실거래가 API 직접 연동 + 모든 수치에 [출처] 명시 |
| **관점 다양성** | 단일 관점, 긍정+부정 섞어서 모호 | CFO(재무) / CSO(전략·리스크) / 투자컨설턴트(적합성) 3인이 역할별 전문 분석 |
| **리스크 분석** | "다만~" 수준의 언급 | CSO가 구조적으로 최소 1개 리스크 반드시 제기 (확증편향 방지) |
| **재무 계산** | 표면 수익률만 언급 | 실질 수익률, 레버리지 수익률, 현금흐름, 세금, IRR 자동 계산 |
| **투자 적합성** | 일반적 조언 | 내 예산·목적·성향에 맞는 구체적 대안과 포트폴리오 자문 |
| **시뮬레이션** | 없음 | Monte Carlo 시뮬레이션, 민감도 분석, 스트레스 테스트 |
| **대안 비교** | "목적에 따라 달라질 수 있습니다" | 성수·마곡 등 대안 권역 수익률·리스크 정량 비교 |
| **정리·후속** | 없음 (대화 종료 시 사라짐) | 비서실장이 결정사항 + Next Action Plan 자동 생성 |

\---

## 차별점 3줄 요약 (발표용)

1. **실거래 데이터 직접 연동 + 출처 명시** — ChatGPT는 "~로 알려져 있습니다"이지만, 우리는 국토교통부 API에서 가져온 실거래가에 [출처]를 달아 신뢰할 수 있는 숫자만 말한다.

2. **CFO·CSO·투자컨설턴트의 영역별 전문 토론** — ChatGPT는 한 명이 긍정·부정을 섞어 모호하게 답하지만, 우리는 재무/전략/적합성 3인이 각자 영역에서 전문적으로 분석하고 건설적으로 충돌한다.

3. **비서실장 자동 회의록** — ChatGPT는 대화가 끝나면 수다로 끝나지만, 우리는 결정사항과 Next Action이 자동으로 정리되어 실제 투자 판단으로 이어진다.

## 발표 시 대응 스크립트

**질문**: "이거 ChatGPT에 역할 나눠서 물어보면 되지 않나요?"

**답변**:
> "좋은 질문입니다. 실제로 해봤습니다. (비교표 보여주며) ChatGPT에 같은 질문을 하면 '수익률 3%는 낮은 편입니다, 다만 시세차익을 기대할 수 있습니다' 이렇게 답합니다. 출처가 없고, 리스크 분석이 얕고, 내 상황에 맞는 자문이 없습니다.
>
> 저희 시스템은 세 가지가 다릅니다. 첫째, 국토교통부 실거래가 API에서 데이터를 가져와 모든 수치에 출처를 답니다. 둘째, CFO·CSO·투자컨설턴트 세 명이 각자 영역에서 전문적으로 분석하는데, 특히 CSO는 반드시 리스크를 지적하도록 설계해서 확증편향을 방지합니다. 셋째, 대화가 끝나면 비서실장이 결정사항과 Next Action을 자동으로 정리해줍니다. ChatGPT는 좋은 대화를 하지만 결정을 도와주진 않습니다."
