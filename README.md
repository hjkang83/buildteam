# 말하는 비서실 (TalkFile)

> "회의실에서 대화하듯 편하게 말씀만 하세요. 당신의 한마디에 소집된 전문가 팀원들이 함께 머리를 맞대고 생각의 진도를 뽑아드립니다."

---

## 한 줄 요약

STT-LLM-TTS 기반으로 사전 구성된 AI 전문가 팀과 **음성으로 회의**하는 대화형 업무 도구

---

## 이 프로젝트가 해결하는 문제

혼자 일하는 1인 창업가, 소규모 팀 리더는 중요한 결정을 내릴 때 "같이 머리 맞대고 얘기할 사람"이 없다. ChatGPT가 있지만 텍스트를 치고, 프롬프트를 짜고, 먼저 물어봐야 하는 건 또 다른 종류의 노동이다.

실제 회사에서는 회의로 일한다. 말로 지시하고, 보고를 듣고, 토론하며 결정한다. 하지만 지금의 AI 서비스는 여전히 "글"에만 포커스를 두고 있다.

**말하는 비서실**은 이 간극을 메운다. 목소리만으로 회의를 시작하면, 세팅된 전문가 에이전트들이 실시간으로 피드백, 반론, 조언을 던지며 생각을 구체화해준다.

---

## 핵심 기능

### 1. 실시간 음성 회의
사용자가 말하면 AI 전문가들이 음성으로 즉시 응답한다. 추임새, 역질문 등 실제 사람과 대화하는 것 같은 자연스러운 티키타카를 유지한다.

### 2. 입체적 전문가 팀 구성

| 에이전트 | 역할 | 말투 예시 |
|----------|------|----------|
| A (실무형) | 데이터 기반 현장 감각 | "대표님, 데이터상으론 이렇습니다만..." |
| B (레드팀) | 반론과 리스크 지적 | "저는 동의하기 어렵네요. 리스크 하나만 짚죠." |
| C (멘토) | 본질과 철학 중심 조언 | "긴 호흡으로 본다면 결국..." |
| 서기 | 중립적 정리와 액션 도출 | "지금 논의를 정리해 보면..." |

### 3. 자동 회의록 & 넥스트 액션
모든 대화가 실시간 스크립트로 기록되고, 회의 종료 후 핵심 요약 + 다음 할 일(Next Action Plan)이 자동 생성된다.

### 4. 지식 선순환 파이프라인
회의록이 옵시디언에 마크다운으로 자동 저장되고, 다음 회의 때 RAG 데이터로 자동 로드되어 대화의 연속성이 유지된다.

## 기술 스택

| 모듈 | 기술 | 역할 |
|------|------|------|
| STT | Deepgram Nova-2 | 음성 → 텍스트 변환 |
| LLM | GPT-4o-mini / Claude Opus | 대화 생성 및 추론 |
| TTS | OpenAI TTS-1 / ElevenLabs | 텍스트 → 음성 변환 |
| RAG | Pinecone | 전문 지식 검색 |
| Orchestrator | Pipecat (오픈소스) | 대화 흐름 제어, 말 끊기 처리 |
| Frontend | 웹 기반 (React) | 에이전트 시각화, 실시간 스크립트 |
